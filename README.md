## Getting Started

### Prerequisites

- Python 3.x
- Jupyter Notebook
- pandas
- numpy
- matplotlib

### Install the necessary libraries:

```sh
pip install pandas numpy matplotlib
```

Clone the repository:
```sh
git clone https://github.com/amaan-igs/data-analytics.git
```

Navigate to the project directory:
```sh
cd infolder
```

Start Jupyter Notebook:
```sh
Jupyter Notebook
```
